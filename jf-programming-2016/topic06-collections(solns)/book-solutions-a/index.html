Exercise 2

Modify the generatePin method so that a PIN cannot comprise 4 consecutive digits in ascending order.

For example 3456 would be disallowed. However 6543 would be acceptable.

Exercise 2 Solution

Approach: ensure that second digit is not 1 greater than 1st.

Remaining digits may be generated without checks.

The algorithic approach could be as follows:

  • Generate the first PIN digit
  • Do-while loop
    • Generate the second digit
  • While second digit 1 greater than 1st regenerate
  • Concatenate 2nd digit to PIN
  • Generate remaining digits.
  • Concatenate each successive digit to PIN

Here is an implementation:

  • For convenience move Math.random to a method randomByte in line with DRY principle.
    public static byte randomByte()
    {
        return (byte)(Math.random()*9 +1);
    }

    public static String generatePin(int nmrPinDigits)
    {
        String pin = new String();

        //first digit
        byte firstDigit = randomByte();
        pin += firstDigit;

        //second digit
        byte thisDigit;
        do
        {
            thisDigit = randomByte();

        } while((thisDigit - firstDigit) == 1);
        pin += thisDigit;

        //remaining digits
        for (int i = 3; i <= nmrPinDigits; i += 1)
        {           
            pin += randomByte();
        }
        return pin;
    }

Here is a method to validate this PIN:

    public static boolean isValid(String pin)
    {
        for(int i = 1; i < pin.length(); i += 1)
        {
            int prevChar = pin.charAt(i-1);
            int thisChar = pin.charAt(i);
            if ((thisChar - prevChar) != 1)
                return true;
        }
        return false;
    }

Exercise 3

In this exercise we define a PIN as being invalid if it comprises consecutive ascending digits (example 45678) otherwise the PIN is valid.

Write a method to test whether a PIN is valid or invalid.

  • Use the signature: public static boolean isValid(String pin)
    • The method should return false if an invalid PIN is detected, else true.

Exercise 3 Solution

    /**
     * Checks if all PIN digits in ascending order
     * If in ascending order the PIN is invalid
     *
     * @param  a string representing the PIN
     * @return true if the PIN is valid else false
     */
    static boolean isValid(String pin)
    {
        for (int i = 1; i < pin.length(); i++)
        {   
          // If the current digit is not one greater 
          // than previous digit, then the pin will be valid.
          if ((pin.charAt(i) - pin.charAt(i-1)) != 1)
          {
            return true;
          }
        }
        return false;
    }

      //Remove key-customer pair from map

Exercise 7

Complete the skeleton code in Figure 2 below so that when the method test() is invoked the matrix shown in Figure 3 is printed.

Exercise 7 Solution

public class TwoDArrays {

  public static final int NUM_ROWS = 10;
  public static final int NUM_COLS = 10;

  static int[][] a;

  public static void generateMatrix()
  {   
    a = new int[NUM_ROWS][NUM_COLS];

    for (int row = 0; row < NUM_ROWS; row += 1)  
    {
      for (int col = 0; col < NUM_COLS; col += 1)  
      {
        a[row][col] = (row + 1)*(col + 1);
      }
    }

  }

  public static void print(int[][] a)
  {
    for (int row = 0; row < NUM_ROWS; row += 1)
    {
      for (int col = 0; col < NUM_COLS; col += 1)
      {
        int val = a[row][col];
        if (val < 10)// for purpose of vertical alignment print leading zero
        {
          System.out.print("0" + a[row][col] + " ");
        }
        else
        {
          System.out.print(a[row][col] + " ");
        }
      }
      System.out.println();
    }
  }
  public static void test()
  {
    generateMatrix();
    print(a);
  }
}

import java.util.HashSet;
import java.util.Iterator;

public class DemoHashSet
{

  public static void demo()
  {
    //Declare and initialize a HashSet
    HashSet<String> contacts = new HashSet<String>();

    //Add this array of strings
    String[] strings = {"DCU", "DCU", "WIT", "CIT"};
    for (String string : strings)
      contacts.add(string);

    //Declare an iterator and use it to traverse the set
    //Print out each element
    //Observe duplicates not present
    Iterator<String> it = contacts.iterator();
    while (it.hasNext())
    {
      String element = it.next();
      System.out.print(element + " ");
    }
    System.out.println();
    //Remove "DCU"
    contacts.remove("DCU");

    //Iterate the set
    //Print out elements
    //Verify element has been removed  
    Iterator<String> it2 = contacts.iterator();
    while (it2.hasNext())
    {
      String element = it2.next();
      System.out.print(element + " ");
    }
    System.out.println();
  }

}
