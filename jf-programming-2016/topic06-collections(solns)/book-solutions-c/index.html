Exercise 1

Validate the constructor parameter shift, ensuring only numbers within a valid range are accepted.

private boolean isValid(int shift)

Exercise 1 Solution

    /*
     * @param The integer value to be validated.
     * @return Returns true if shift in range (0, 26)
     */
    private boolean isValid(int shift)
    {
        if(shift > 0 && shift < 26)
        {
            return true;
        }
        return false;
    }

The validation could be (naively) employed as follows:

    public CaesarCipher(int shift)
    {
        if (isValid(shift))
        {
            this.shift = shift;
        }
        else
        {
            System.out.println("Enter valid shift: positive number in range 1 to 25 inclusive");
        }
    }

Validate both the message and cipher text strings so as to ensure all characters are upper case letters.

private boolean isUpperCase(String string)

Introduce the method into your code and test by inputting, first valid data, then invalid data.

Exercise 2 Solution

    /*
     * Permitted characters are A-Z.
     * 
     * @see https://en.wikipedia.org/wiki/ASCII#ASCII_printable_characters
     * @param string
     */
    private boolean isUpperCase(String string)
    {
        char[] chars = string.toCharArray();
        for (char ch : chars)
        {
            if (!Character.isUpperCase(ch))
            {
                return false;
            }
        }
        return true;
    }

The validation could be (naively) employed as follows:

    /**
     * Encrypt a plain text string using the Caesar Cipher
     * @param messageText The message text (plain text)
     * @return Returns the cipher text string
     */
    public String encrypt(String messageText)
    {
        if (!this.isUpperCase(messageText))
        {
            System.out.println("Input string must comprise only alphabet characters");
            return "";
        }

        char[] chars = messageText.toCharArray();
        char[] encryptedChars = new char[chars.length];
        for (int i = 0; i < chars.length; i += 1)
        {
            encryptedChars[i] = encrypt(chars[i]);
        }
        return new String(encryptedChars);
    }

Exercise 4: Substitution Cipher

The Caesar Cipher is categorized as a substitution cipher. Here we shall develop a substitution cipher in which the set of valid characters comprise the printable ASCII set.

The ASCII table reveals this set to be in the range [32, 126], 32 being the space and 126 the tilde (~).

Still in the encryption project, develop a class named SubstitutionCipher, mining as much code as you can from the Caesar Cipher.

Here is starter code. 

public class SubstitutionCipher
{

  // The number of characters to shift to encrypt
  int shift;
  // Defines the printable range of ASCII characters
  private int minAscii = 32;
  private int maxAscii = 126;

  public SubstitutionCipher(int shift)
  {
    this.shift = shift;
  }
}

Exercise 4 Solution

/*
 * Substitution cipher does not have a key. Simply shift by a number
 * of characters down the alphabet. For example if shift is 1
 * then A plaintext becomes B cipher text.
 * Plaintext may comprise printable ASCII characters.
 * The shift is a positive number. If a negative shift is entered it
 * its absolute value is taken as the shift.
 */
public class SubstitutionCipher
{

    // The number of characters to shift
    int shift;
    // Defines the printable range of ASCII characters
    private int minAscii = 32;
    private int maxAscii = 126;

    /**
     * SubstitutionCipher constructor
     * 
     * @param shift A positive digit by which to shift plain text.
     */
    public SubstitutionCipher(int shift)
    {

        this.shift = Math.abs(shift);
    }


    public String encrypt(String messageText)
    {
        byte[] bytes = messageText.getBytes();
        byte[] encryptedBytes = new byte[bytes.length];
        for (int i = 0; i < bytes.length; i += 1)
        {
            encryptedBytes[i] = encrypt(bytes[i]);
        }
        return new String(encryptedBytes);
    }

    private byte encrypt(byte byt)
    {
        int encryptedByte = byt + shift;
        if (encryptedByte > maxAscii)
        {
            encryptedByte = minAscii + encryptedByte % maxAscii - 1;
        }
        return (byte) encryptedByte;
    }

    public String decrypt(String cipherText)
    {
        byte[] bytes = cipherText.getBytes();
        byte[] decryptedBytes = new byte[bytes.length];
        for (int i = 0; i < bytes.length; i += 1)
        {
            decryptedBytes[i] = decrypt(bytes[i]);
        }
        return new String(decryptedBytes);
    }

    private byte decrypt(byte byt)
    {
        int range = maxAscii - minAscii;
        int decryptedByte = byt + range - shift + 1;
        if (decryptedByte > maxAscii)
        {
            decryptedByte = minAscii + decryptedByte % maxAscii - 1;
        }
        return (byte) decryptedByte;
    }

    public static void unitTest_1()
    {
        String messageText = "Was I ever so clever as when I was 15?";

        for (int shift = 1; shift <= 25 ; shift += 1)
        {
            SubstitutionCipher caesarCipher = new SubstitutionCipher(shift);
            String cipherTxt = caesarCipher.encrypt(messageText);
            String dcryptTxt = caesarCipher.decrypt(cipherTxt);
            if (!dcryptTxt.equals(messageText))
            {
                System.out.println("Oh oh: problem at GCHQ");
            }
            else
            {
                System.out.println("Successful encryption-decryption using shift :" + shift);
            }
        }
    }

    public static void unitTest_2()
    {
        SubstitutionCipher cc = new SubstitutionCipher(3);
        String messageText = "Was I ever so clever as when I was 15?";
        System.out.println("Message text:  " + messageText);
        String cipherText = cc.encrypt(messageText);
        System.out.println("Cipher Text: " + cipherText);
        System.out.println("Message Text:  " + cc.decrypt(cipherText));

    }
}

Exercise 5

In the SubstitutionCipher write a method to validate the input string (message or cipher). A valid string shall contain only printable characters. Alert the user by printing a warning message if an invalid string is presented.

  private boolean isASCIIPrintable(String string)

Introduce the method at appropriate locations in SubstitutionCipher.

Exercise 5 Solution

    /**
     * Permitted characters are ASCII printable characters
     * 
     * @see https://en.wikipedia.org/wiki/ASCII#ASCII_printable_characters
     * @param string
     */
    private boolean isASCIIPrintable(String string)
    {
        byte[] bytes = string.getBytes();
        for (byte byt : bytes)
        {
            if (byt < minAscii || byt > maxAscii)
            {
                return false;
            }
        }
        return true;
    }
    public String encrypt(String messageText)
    {
        if (!this.isASCIIPrintable(messageText))
        {
            System.out.println("Input string must comprise only printable ASCII characters");
            return "";
        }

        byte[] bytes = messageText.getBytes();
        byte[] encryptedBytes = new byte[bytes.length];
        for (int i = 0; i < bytes.length; i += 1)
        {
            encryptedBytes[i] = encrypt(bytes[i]);
        }
        return new String(encryptedBytes);
    }
