Guidelines

Introduction

This assignment comprises Question 1 (Q1), Question 2 (Q2) and Question 3 (Q3).

Submit your completed assignment as a single zip file only (no internal zip files):

  • assignment-2.zip

Please do ensure that the code you submit (even if all the specified tasks have not been completed):

  • Compiles without errors.
  • Runs to completion in the various tests without thowing exceptions or crashing.
  • For Q1 and Q2 it is sufficient only to ensure correct styling such as indentation, horizontal and whitespace positioning and so on. It is not necessary to provide any additional commenting unless you consider this might be desirable should self-documentation prove inadequate.
  • Fully document Q3, the Vigenere Cipher. The API is already documented. So it only remains to document the implementation. Use the same approach to documenting private methods and variables as has been adopted for the API.

Sorting is employed in both questions.

  • You are not required to develop a sorting algorithm.
  • You are, however, required to modify the selection sort algorithm already encountered in the labs in the case of Q2 (Spacebook).
  • The selection sort algorithm used in Q1 (Auction) has already been fully adapted from the lab version that sorts integers to a version that sorts Bid objects. You are not required to make any additional changes here.
  • In Q2 (Spacebook) you are required to perform similar modifications to sort Message and String objects using the simple selection sort algorithmic pattern.
