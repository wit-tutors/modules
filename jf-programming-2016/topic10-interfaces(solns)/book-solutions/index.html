public class Cone implements Measurable
{
    int height;
    int radius;

    public Cone(int height, int radius) 
    {
        this.height = height; 
        this.radius = radius;
    }

    public double volume()
    {
        return areaBase()*height/3;
    }

    /**
     * Calculates the surface area of right circular cone (including base).
     * @return total surface area right circular cone (includes base)
     */
    public double surfaceArea()
    {
        double slantHeight  = Math.hypot(radius, height);
        double areaTop      = Math.PI*radius*slantHeight;
        return areaTop + areaBase();
    }

    private double areaBase()
    {
        return Math.PI*radius*radius;
    }

    public double getMeasure()
    {
        return surfaceArea();
    }
}

public class Circle implements Measurable

    int radius;

    public Circle(int radius)

        this.radius = radius;

    public double getMeasure()

        return area();

11.1 Selection Sort

Here is the Comparable interface:

public interface Comparable
{
  int compareTo(Comparable comparable);
}
public class SelectionSort
{ 
  public static void sort(Comparable[] a)
  {
    for (int i = 0; i < a.length; i += 1)
      for (int j = i; j < a.length; j += 1)
        if (a[i].compareTo(a[j]) < 0)
          exchange(a, i, j);
  }

  private static void exchange(Comparable[] a, int i, int j)
  {
    Comparable val = a[i];
    a[i] = a[j];
    a[j] = val;
  }

}

11.2 Message and Circle classes


public class Message implements Comparable
{
  String message;

  public Message(String message)
  {
    this.message = message;
  }

  @Override
  public int compareTo(Comparable comparable)
  {
    Message msg = (Message)comparable;
    return msg.message.compareTo(message);
  }

  public String toString()
  {
    return message + "\n";
  }
}
/**
 * Observe that compareTo has been fully implemented here.
**/
public class Circle implements Comparable
{
    int radius;

    public Circle(int radius)
    {
        this.radius = radius;
    }

    @Override
    public int compareTo(Comparable comparable)
    {
      Circle c = (Circle)comparable;
      if (c.radius < radius)
        return -1;
      if (c.radius > radius)
        return 1;
      return 0;
    }

    @Override
    public String toString()
    {
      return "radius: " + String.valueOf(radius) + "\n";
    }
}

11.4 Why does single print method work?

  • The toString methods have been overridden in both Message and Circle. Therefore executing a statement such as print(circle) where circle is a Circle object will output a textual representation of circle (by invoking toString).
  • consider the statement print(messages) in the main method:
    • messages is a reference to an array of Message objects: Message[] messages.
    • the signature of print is : print(Comparable[] comparables)
      • although the actual parameter in print is a Message[] type, the formal paramater is Comparable[] type. Recall that Message implements Comparable and thus it is allowable that an array of Comparable types references an array of Message types.
