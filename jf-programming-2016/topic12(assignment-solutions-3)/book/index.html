Readme

This is a sample solution set to the third and final Assignment which is based on materials covered in the entire course but with particular emphasis on the modules Designing classes (Spacebook) and Inheritance.

Also provided is a demonstration project that develops a generic mergesort algorithm that may be utilized to sort specific attributes of objects of different classes.

A sample project demonstrating the use of the mergesort algorithm to sort a range of classes is also provided:

  • Included is a generic version of mergesort
    • Filenames: MyCollections_v0.java & MyCollections_v1.java
  • Two test programs are also included which demonstrate use of the generic mergesort:
    • TestSortMessages
    • TestSortUsers

  • You might consider studying the files utils/MyCollections_v0.java and utils/MyCollections_v1.java.

    • Both contain generic versions of mergesort
      • They differ in that version v1 has a static sort method, unlike v0.

  • You can easily verify that, for example, v1 works in the spacebook assignment project.

    • Simply add the file to the util folder, add suitable import statements and replace calls such as:
    Collections.sort(inbox, new MessageFromComparator());

with

    MyCollections_v1.sort(inbox, new MessageFromComparator());
