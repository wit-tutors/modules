Lab-01

On completion of this lab you should:- be familiar with the PDE (Processing Development Environment).- be able to use methods to draw basic shapes and add colour to these shapes.
Lab-02

On completion of this lab you should:- be able to write code to animate simple drawings.- be aware of the system variables in the PDE and be able to use them effectively.- understand primitive data types and be able to use them in non-complex problems.
Code to Download

On completion of this lab you should:- have downloaded code used in class.- understand the code used.
Lab-03

On completion of this lab you should be able to code animated drawings using the following constructs:- conditional statements- relational operators- logical operators- variables- mouse events- key events
Lab-04

On completion of this lab you should understand variable scope and be able to code static drawings using for and while loops.
Lab-05
  • On completion of this lab you should be able to use methods to handle mouse events and also be able to write your own methods.- There are no exercises for this lab; instead we suggest you use the lab time to work on your assignment.
Lab-06
  • On completion of this lab you should be able to write more sophisticated methods, particularly using parameters and returning data. You will also learn how to use the String methods.
Lab-07

On completion of this lab you should:- be able to write code with more String methods- be able to write a class and an object of the class.- be able to write and call class contructors.
Lab-08

On completion of this lab you should:- be able to write methods for a class and call those methods- be able to write getters and setters for each field.- be able to write code to implement given validation rules in setters and constructors- start to understand how we protect the data in an object via class code.

Lab-9a

On completion of this lab you should:- be able to write Swing components- be able to write JOptionPane methods- be able to produce message and dialog boxes
Lab-9b

On completion of this lab you should:- be able to declare and construct an array- be able to write loops involving arrays- be able to read values in from the keyboard into an array using JOptionPane methods.

Lab-9c

On completion of this lab you should:- be familiar with the game of pong. - be able to write methods for a class and call those methods.- be able to write getters and setters for each field.- be able to write code to implement given validation rules in setters and constructors.- start to understand how we protect the data in an object via class code. In this lab, you will work on PongGameV1_0 and PongGameV2_0.
Lab-10a

On completion of this lab you should:- be very familiar with the game of pong. - be able to include collision detection code for the Ball and Paddle class.- be able to maintain lives lost and scores for the game.- be able to maintain tournament data.In this lab, you will work on PongGameV3_0, PongGameV4_0 and PongGameV5_0.
Lab-10b

On completion of this lab you should:- - be familiar with the Player class. - be familiar with how arrays are used in the Player class. - be familiar with how the Player class is incorporated into the game.- be able to understand and call simple data analysis algorithms.- be able to switch output from console to dialog boxes when appropriate. In this lab, you will work on PongGameV5_0, PongGame7_0 and PongGameV8_0.
Lab-10c

On completion of this lab you should:- be familiar with the pythagorean theorem for collision detection.In this lab, you will work on PongGameV9_0.
Formative Assessment

The details of your formative assessment are listed in this area. The marks you get from this assessment will NOT go towards your final module mark but will act as a learning tool for attempting and submitting an assignment.
Assignment-1

Overview of assignment - This is your first assignment. - You are asked to write a working animation. - You will then be asked to present your code during your lab the week following submission.
Assignment-2

Overview of assignment - This is your second assignment. - You are asked to write a working animation. - You will be asked to interview on your code during study week.
