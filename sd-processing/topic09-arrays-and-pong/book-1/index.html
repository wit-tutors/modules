Using Swing - JoptionPane

  • To use JOptionPane, we must import the Swing package
import javax.swing.*;

Simple Message Output

In this step, you will use JOptionPane to write a simple message.

Create a new Processing sketch in your workspace and call it JOption.

Include the following code in your sketch (don't cut and paste it, type it in): Add in the code to the import statement above:

JOptionPane.showMessageDialog(null,
